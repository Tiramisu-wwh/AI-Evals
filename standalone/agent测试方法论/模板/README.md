## 使用顺序

1. 先用 [跨角色对齐清单模板](./00-通用/跨角色对齐清单模板.md) 组织需求、技术方案、测试用例评审。
2. 产品设计阶段填写 [成功标准模板](./G1-产品设计评审/成功标准模板.md)。
3. 技术方案阶段补齐 G2 目录中的方案、工具边界和可观测性模板。
4. 研发实现阶段补齐 G3 目录中的实现检查和联调记录。
5. 测试评估阶段使用 G4 目录中的样本、评分器、轮次结果和坏例模板。
6. 质量和安全交付阶段分别使用 G5、G6 目录中的验收模板。
7. 上线后使用 G7 目录中的周运行、坏例回流和阶段复盘模板。

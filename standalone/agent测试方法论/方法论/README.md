1. 先读 [Agent评估体系建设初稿](../Agent评估体系建设初稿.md)，确认阶段、责任人和交付物。
2. 再读 [Agent测试全流程方法论](../Agent测试全流程方法论.md)，确认测试评估分册内容。
3. 再读 [参考文章吸收说明](../参考文章吸收说明.md)，确认外部文章的吸收范围和边界。
4. 对专业词不熟时，直接查 [专业词汇附录](./专业词汇附录.md)。
